# 서비스 기획서

## 1. 프로젝트 개요

youtube-trend-intelligence는 빠르게 변화하는 트렌드를 쉽고 직관적으로 파악할 수 있도록 돕는 시계열 기반 트렌드 분석 서비스이다.

이 서비스의 핵심은 단순히 현재 인기 있는 콘텐츠를 보여주는 것이 아니라, 시간에 따라 만들어지고 확산되는 흐름을 분석해 사용자가 트렌드의 방향과 변화를 빠르게 이해할 수 있도록 하는 데 있다.

초기에는 공개 API를 제공하는 유튜브를 데이터 소스로 활용하여 영상, 채널, 키워드 단위의 변화 흐름을 분석한다.
단, 초기 버전은 유튜브 전체를 완전하게 추적하는 서비스가 아니라 YouTube Data API로 반복 수집 가능한 공개 데이터 표본을 기반으로 동작한다.

## 2. 문제 정의

온라인 트렌드는 매우 빠르게 변한다.
특정 키워드, 특정 영상, 특정 채널은 짧은 시간 안에 급부상할 수 있고, 과거에 이미 소비된 콘텐츠가 다시 주목받는 경우도 빈번하게 발생한다.

하지만 이러한 변화를 직접 파악하려면 다양한 콘텐츠와 반응을 지속적으로 확인해야 하며, 이는 많은 시간과 피로도를 요구한다.

특히 트렌드 파악이 중요한 사용자들은 단순한 인기 목록만으로는 충분한 정보를 얻기 어렵다.
이들에게 필요한 것은 다음과 같은 정보이다.

- 지금 무엇이 뜨고 있는가
- 무엇이 새롭게 올라오고 있는가
- 어떤 채널이 빠르게 성장하고 있는가
- 어떤 키워드가 반복적으로 등장하고 있는가
- 어떤 흐름이 숏폼에서 강한가, 어떤 흐름이 롱폼에서 강한가
- 어떤 대상이 앞으로 더 확산될 가능성이 있는가

따라서 빠르게 변화하는 트렌드를 구조적으로 분석하고, 이를 한눈에 파악할 수 있게 제공하는 서비스가 필요하다.

본 프로젝트는 이러한 문제를 해결하기 위해 시작되었다.

## 2.1 현실적인 제품 범위

초기 단계에서 직접 관측 가능한 대상은 다음과 같다.

- 지역 / 카테고리별 인기 영상 목록
- 수집된 인기 영상에 포함된 채널
- 수집된 영상의 제목 / 설명 / 태그에서 추출한 키워드
- 제한된 키워드 검색으로 확장한 최근 영상 후보

직접 관측하기 어려운 대상은 다음과 같다.

- 유튜브 전체에서 막 뜨기 시작한 모든 신규 영상
- 아직 인기 표본이나 검색 표본에 들어오지 않은 신생 채널
- 유튜브의 공식 검색어 순위

따라서 본 서비스는 전수 감시 시스템이 아니라, 공개 API로 확보한 표본을 시계열로 누적해 흐름을 해석하는 분석 도구로 정의한다.

## 3. 프로젝트 목표

본 프로젝트의 목표는 사용자가 빠르게 변화하는 트렌드를 쉽게 파악할 수 있도록 돕는 것이다.

이를 위해 다음을 제공한다.

- 현재 강하게 주목받는 대상
- 최근 빠르게 상승 중인 대상
- 반복적으로 등장하는 키워드
- 과거 콘텐츠의 리바이벌 현상
- 숏폼과 롱폼의 흐름 차이
- 단기적으로 더 확산될 가능성이 있는 대상

궁극적으로는 트렌드 파악이 중요한 사용자들이 더 빠르게 흐름을 읽고, 이를 콘텐츠 기획, 마케팅, 운영 전략 등 실질적인 의사결정에 활용할 수 있도록 돕는 것을 목표로 한다.
이때 제공하는 모든 결과는 "유튜브 전체의 공식 트렌드"가 아니라 "수집 표본 안에서 계산된 트렌드"라는 점을 전제로 한다.

## 4. 주요 타겟 사용자

### 4.1 마케팅팀 / 브랜드 마케터
현재 반응이 커지고 있는 주제와 키워드를 빠르게 파악하고, 캠페인이나 메시지 전략에 반영해야 하는 사용자

### 4.2 콘텐츠 기획자 / 미디어 운영자
최근 반응이 좋은 주제, 포맷, 채널 흐름을 분석해 콘텐츠 아이템과 운영 방향을 결정해야 하는 사용자

### 4.3 인플루언서 / 크리에이터
지금 어떤 채널이 성장하고 있는지, 어떤 키워드와 포맷이 확산되는지 파악해 자신의 콘텐츠 전략에 반영해야 하는 사용자

### 4.4 트렌드 분석이 필요한 실무자
시장 흐름이나 대중 관심사의 변화를 빠르게 읽고, 이를 업무 판단에 반영해야 하는 사용자

### 4.5 일반 사용자
현재 유행하는 흐름을 빠르게 파악하고 싶은 사용자

이 서비스의 주요 타겟은 일반 사용자 전체가 아니라, 트렌드 파악이 업무나 성과와 직접 연결되는 사용자층이다.

## 5. 사용자 시나리오

### 시나리오 1: 마케팅팀
마케팅 담당자가 서비스에 접속한다.
최근 24시간 동안 급상승한 키워드와 채널을 확인하고, 현재 반응이 커지고 있는 주제를 파악해 마케팅 메시지와 캠페인 방향을 조정한다.

### 시나리오 2: 콘텐츠 기획자
콘텐츠 기획자가 최근 7일간 숏폼 영역에서 빠르게 성장한 채널과 반복적으로 등장한 키워드를 확인한다.
이를 통해 현재 반응이 좋은 포맷과 주제를 파악하고 다음 콘텐츠 아이템을 정한다.

### 시나리오 3: 인플루언서 / 크리에이터
크리에이터가 최근 떠오르는 채널과 키워드를 확인한다.
어떤 흐름이 형성되고 있는지 참고하여 업로드 주제와 포맷을 결정한다.

### 시나리오 4: 실무자
실무자가 최근 뜨는 영상과 리바이벌 콘텐츠를 확인한다.
현재의 반응이 단기적 유행인지, 과거 자산이 다시 살아나는 흐름인지 구분해 판단한다.

## 6. 핵심 기능

### 6.1 Hot Video
현재 강하게 주목받고 있는 영상을 제공한다.

### 6.2 Rising Video
최근 짧은 시간 안에 빠르게 상승 중인 영상을 제공한다.
단, 수집 후보군 안에서의 상승으로 정의한다.

### 6.3 Revival Video
오래전에 업로드되었지만 최근 다시 관심이 증가한 영상을 제공한다.
장기 스냅샷이 충분히 쌓인 뒤에만 신뢰도 있게 제공할 수 있다.

### 6.4 Rising Channel
최근 영상 성과와 성장 흐름이 강한 채널을 제공한다.

### 6.5 Breakout New Channel
신생 채널이지만 빠르게 주목받고 있는 채널을 제공한다.
단, 수집된 영상 또는 검색 후보를 통해 관측된 채널에 한한다.

### 6.6 Hot Keyword
반복적으로 등장하고 확산 중인 키워드를 제공한다.
이 키워드는 공식 검색어 랭킹이 아니라 내부 추출 결과다.

### 6.7 Short-form / Long-form Trend
숏폼과 롱폼의 흐름을 분리해 보여준다.

### 6.8 Short_term Prediction
앞으로 더 상승할 가능성이 있는 대상에 대한 예측 정보를 제공한다.

## 7. 서비스가 제공하는 가치

본 서비스는 단순한 인기 콘텐츠 확인 도구가 아니라, 트렌드의 형성과 확산 과정을 분석해 제공하는 데 가치를 둔다.

사용자는 이를 통해 다음과 같은 가치를 얻을 수 있다.

- 트렌드를 빠르게 파악할 수 있다.
- 직접 다양한 콘텐츠를 탐색해야 하는 피로도를 줄일 수 있다.
- 변화 흐름을 데이터 기반으로 이해할 수 있다.
- 실질적인 의사결정에 활용할 수 있다.

## 8. 차별점

이 서비스는 다음 점에서 차별화된다.

- 단순 인기 목록이 아니라 변화량과 흐름 중심 분석
- 영상뿐 아니라 채널 성장 흐름 제공
- 키워드를 직접 추출하여 흐름을 설명
- 숏폼 / 롱폼 분리 분석
- 과거 콘텐츠 리바이벌 탐지
- 단기 확산 가능성 예측

## 9. 초기 데이터 소스 선택 이유

본 서비스의 목적은 트렌드 분석이며, 유튜브는 이를 구현하기 위한 초기 데이터 소스이다.

유튜브를 선택한 이유는 다음과 같다.

- 공개 API를 제공한다.
- 영상, 채널, 키워드 단위의 흐름을 추적하기에 적합하다.
- 트렌드 변화가 빠르게 나타난다.
- 숏폼과 롱폼을 함께 다룰 수 있다.

다만 다음 한계가 있다.

- `search.list` quota 비용이 높아 전수 탐색이 어렵다.
- Shorts를 공식적으로 구분하는 단일 공개 필드가 없다.
- 구독자 수 증감은 공개 스냅샷 기준 추정치에 가깝다.
- 표본에 들어오지 않은 대상은 분석할 수 없다.

향후에는 다른 데이터 소스로 확장 가능하다.

## 10. 최종 서비스 이미지


최종적으로 이 서비스는 사용자가 접속했을 때 현재 어떤 흐름이 뜨고 있고, 무엇이 새롭게 형성되고 있으며, 앞으로 어떤 대상이 더 커질 가능성이 있는지를 빠르게 이해할 수 있는 표본 기반 트렌드 인텔리전스 서비스가 된다.
